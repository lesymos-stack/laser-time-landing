=====================================================
  НАСТРОЙКА ПРИЁМА ЗАЯВОК
  Google Таблица + Telegram уведомления
=====================================================

Время настройки: 10-15 минут
После настройки все заявки с квиза и контактной формы
будут сохраняться в Google Таблицу и приходить в Telegram.


=== ЧАСТЬ 1: СОЗДАНИЕ TELEGRAM БОТА (5 минут) ===

1. Откройте Telegram, найдите бота @BotFather
2. Напишите ему /newbot
3. Придумайте название (например: Лазер Тайм Заявки)
4. Придумайте username (например: lasertime_leads_bot)
5. BotFather даст вам ТОКЕН — скопируйте его
   Выглядит так: 7123456789:AAHfiqksKZ8WmR2zCh45y...

6. Теперь найдите бота @userinfobot
7. Напишите ему /start
8. Он покажет ваш ID — скопируйте число (например: 123456789)
   Это ваш CHAT_ID

9. ВАЖНО: Найдите вашего нового бота в поиске Telegram
   и нажмите /start — без этого бот не сможет вам писать!


=== ЧАСТЬ 2: СОЗДАНИЕ GOOGLE ТАБЛИЦЫ (5 минут) ===

1. Откройте https://sheets.google.com
2. Создайте новую таблицу (кнопка +)
3. Назовите её "Лазер Тайм — Заявки"
4. В меню нажмите: Расширения -> Apps Script
5. Откроется редактор кода
6. Удалите всё что там написано
7. Откройте файл GOOGLE_APPS_SCRIPT.js из папки проекта
8. Скопируйте ВЕСЬ код из этого файла
9. Вставьте в редактор Apps Script

10. В НАЧАЛЕ кода найдите две строки:
    var TELEGRAM_BOT_TOKEN = 'ВСТАВЬТЕ_ТОКЕН_БОТА';
    var TELEGRAM_CHAT_ID = 'ВСТАВЬТЕ_CHAT_ID';

    Замените на ваши данные из Части 1:
    var TELEGRAM_BOT_TOKEN = '7123456789:AAHfiqksKZ8WmR2zCh45y...';
    var TELEGRAM_CHAT_ID = '123456789';

11. Нажмите кнопку "Сохранить" (или Ctrl+S)


=== ЧАСТЬ 3: ПУБЛИКАЦИЯ СКРИПТА (3 минуты) ===

1. В редакторе Apps Script нажмите:
   Развертывание -> Новое развертывание

2. Нажмите шестеренку слева и выберите "Веб-приложение"

3. Заполните:
   - Описание: Прием заявок
   - Выполнять как: Я (ваш email)
   - Доступ: Все (даже анонимные пользователи)

4. Нажмите "Развернуть"

5. Google попросит разрешения — нажмите "Проверить"
   потом "Перейти к проекту" -> "Разрешить"

6. Скопируйте URL вида:
   https://script.google.com/macros/s/AKfycbx.../exec

   ЭТО ВАШ URL ДЛЯ ПРИЁМА ЗАЯВОК


=== ЧАСТЬ 4: ПОДКЛЮЧЕНИЕ К САЙТУ (1 минута) ===

1. Откройте файл js/form.js

2. В самом начале найдите строку:
   const GOOGLE_SCRIPT_URL = '';

3. Вставьте ваш URL между кавычек:
   const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/AKfycbx.../exec';

4. Сохраните файл

5. Если сайт на Vercel — закоммитьте изменения и запушьте:
   git add js/form.js
   git commit -m "Подключить Google Sheets"
   git push


=== ПРОВЕРКА ===

1. Откройте сайт
2. Пройдите квиз и отправьте тестовый номер
3. Проверьте:
   - В Google Таблице появился лист "Заявки с квиза" с данными
   - В Telegram пришло уведомление от бота
4. Отправьте сообщение через форму "Написать нам"
5. Проверьте:
   - В таблице появился лист "Обратная связь"
   - В Telegram пришло уведомление


=== ВОЗМОЖНЫЕ ПРОБЛЕМЫ ===

Проблема: Заявки не приходят
Решение: Проверьте что URL правильно вставлен в form.js
         и что скрипт опубликован с доступом "Все"

Проблема: В Telegram не приходит
Решение: Убедитесь что вы написали /start вашему боту
         и что CHAT_ID и TOKEN правильные

Проблема: Google просит разрешения снова
Решение: Это нормально при первом запуске, нажмите "Разрешить"

Проблема: Ошибка "Script function not found: doPost"
Решение: Убедитесь что вставили ВЕСЬ код из файла GOOGLE_APPS_SCRIPT.js
