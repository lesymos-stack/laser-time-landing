# Лендинг-квиз: Лазерная эпиляция

Одностраничный мобильный лендинг с квизом для генерации лидов в салон лазерной эпиляции (г. Анапа).

## Как запустить

Открыть `index.html` в браузере. Сервер не требуется.

## Структура проекта

```
index.html          — Главная страница лендинга с квизом
thank-you.html      — Страница «Спасибо» после отправки формы
privacy.html        — Политика конфиденциальности
css/
  reset.css         — Сброс стилей
  main.css          — Основные стили лендинга
  quiz.css          — Стили квиза и формы захвата
js/
  quiz.js           — Логика квиза (вопросы, прогресс, ответы)
  form.js           — Маска телефона, валидация, отправка формы
  analytics.js      — События Яндекс.Метрики
assets/images/      — Папка для фото салона и результатов
```

## Что нужно заменить перед публикацией

1. **Фото** в `assets/images/` — реальные фото салона и результатов «до/после»
2. **Видео-сторис** мастера — снять и подключить вместо аватарки-заглушки
3. **ИНН и название салона** — в футере `index.html` и `thank-you.html`, а также в `privacy.html`
4. **ID Яндекс.Метрики** — в `js/analytics.js` (переменная `YM_ID`)
5. **Webhook для отправки формы** — в `js/form.js` заменить `https://httpbin.org/post` на реальный endpoint
6. **Адрес салона** — в подвале `index.html` и ссылку на Яндекс.Карты
7. **Цены** — в `thank-you.html` и в `js/quiz.js` (объект `prices` в методе `fillPricePreview`) — проверить актуальность
8. **Ссылку на онлайн-запись** — на кнопке «Выбрать удобное время» в `thank-you.html`
