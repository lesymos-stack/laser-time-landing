Реализуй класс `QuizManager` со следующими методами:
- `init()` — рендерит первый вопрос, привязывает события

- `showHint(text)` — показывает всплывающую подсказку под вариантами
- `nextQuestion()` — переход к следующему вопросу с анимацией слайда
- `updateProgress()` — обновляет полосу прогресса (шаг X из 4)
- `showLeadForm()` — скрывает квиз, показывает форму захвата
- `getAnswers()` — возвращает объект с ответами пользователя
