# 🎯 Проект: Лендинг-квиз для салона лазерной эпиляции (Анапа)

## Описание проекта
Одностраничный мобильный лендинг с встроенным квизом для генерации лидов в салон лазерной эпиляции.
Главная цель: посетитель проходит 4 вопроса → оставляет телефон → получает расчёт цены в Telegram.

## Стек технологий
- **HTML/CSS/JS** — чистый код без фреймворков (для максимальной скорости загрузки)
- **Шрифт** — Inter (Google Fonts)
- **Иконки** — Lucide Icons (CDN)
- **Анимации** — CSS transitions, без тяжёлых библиотек
- **Адаптив** — Mobile First (320px → 768px → 1200px)

## Структура файлов проекта

    └── images/         # Заглушки для реальных фото (placeholder)

## Целевая аудитория
- Женщины 22–45 лет, г. Анапа (постоянные жители, НЕ туристы)
- Основное устройство: смартфон
- Страхи: боль, непонятная стоимость, количество сеансов

## Ключевые требования к коду
- Скорость загрузки < 3 сек на мобильном (без тяжёлых зависимостей)
- Все кнопки минимум 48px высотой (удобство тапа)
- Шрифты минимум 16px
- Плавные CSS-анимации переходов между экранами квиза
- Форма: поле телефона с маской +7 (___) ___-__-__
- После отправки формы: редирект на thank-you.html

## Команды разработки
- Открыть: просто открыть index.html в браузере
- Проверить мобайл: DevTools → Toggle Device Toolbar → iPhone 14

## Важные бизнес-правила (не менять без согласования)
1. Оффер на кнопке: «пробный сеанс по специальной цене» — НЕ «скидка 50%»
2. Квиз: ровно 4 вопроса, не больше
3. Форма: только Telegram как основной канал связи
4. Страница Спасибо: один сценарий — «администратор напишет в Telegram»
5. Цены в вилке должны совпадать с реальным прайсом салона
